# Scrollytics

A Google Analytics jQuery plugin for measuring page scrolling percentage

```HTML
    
    //After GA Universal tag and jQuery script tag:
    <script src="scrollytics.js"></script>
    <script>
        jQuery.scrollytics({
            track: [0, 50, 75, 100],
            debug: true,
        });
    </script>

```
